Warp is a Rust-based terminal that has evolved into a full agentic development environment. It supports a built-in coding agent and integrates with external CLI agents (Claude Code, Codex, Gemini CLI, etc.). The repo was open-sourced with OpenAI as founding sponsor, with agentic management workflows powered by GPT models. The UI framework is MIT-licensed; the rest is AGPL v3.

## Key Features

- Built-in coding agent alongside bring-your-own-agent support (Claude Code, Codex, Gemini CLI)
- Oz agents that triage issues, write specs, implement changes, and review PRs (visible at build.warp.dev)
- Wasm-compiled terminal embeddable in the browser for viewing live agent sessions
- Drive for shared context, plus Warp Code for AI-assisted coding workflows
- Community contribution workflow with `ready-to-spec` and `ready-to-implement` issue labels

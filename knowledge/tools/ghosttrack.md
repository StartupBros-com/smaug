GhostTrack is an OSINT and information-gathering tool written in Python that accepts a phone number, IP address, or username and returns geolocation, carrier data, and cross-platform account presence. It runs on Linux and Termux (Android) with minimal setup — clone, pip install requirements, run the script. With 12,000+ stars it has wide adoption in the security community despite being a dual-use surveillance tool.

## Key Features

- Phone number lookup: carrier info, country, line type, and registration footprint across platforms
- IP geolocation with optional combo mode using the companion Seeker tool for real-world location narrowing
- Username enumeration: searches target usernames across multiple social media platforms
- Runs on standard Linux (apt) and Termux without special dependencies

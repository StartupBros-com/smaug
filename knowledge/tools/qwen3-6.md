Qwen3.6-27B is Alibaba's dense open-source model released in April 2026. At 27B parameters it surpasses the much larger Qwen3.5-397B-A17B MoE model across all major coding benchmarks, matches Claude 4.5 Opus on Terminal-Bench 2.0, and beats it on SkillsBench, GPQA Diamond, MMMU, and RealWorldQA. It supports both thinking and non-thinking modes and is released under Apache 2.0. The model fits in 24GB VRAM at Q4 quantization — the RTX 3090 / 4090 consumer card class — making it the strongest local agentic coding model available on a single consumer GPU as of its release.

## Key Features

- 27B dense architecture, fits on a single 24 GB consumer GPU (Q4 quant)
- Outperforms Qwen3.5-397B-A17B MoE (15x larger) on all major coding benchmarks
- Matches Claude 4.5 Opus on Terminal-Bench 2.0 (59.3); beats it on SkillsBench, GPQA Diamond, MMMU, RealWorldQA
- Supports thinking and non-thinking inference modes
- Apache 2.0 license — fully open weights
- Strong multimodal reasoning alongside text

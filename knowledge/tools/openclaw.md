OpenClaw is a self-hosted personal AI assistant that runs on your own devices and responds on the messaging channels you already use. You run a local gateway (control plane) and the assistant shows up in WhatsApp, Telegram, Slack, Discord, iMessage, Signal, IRC, Matrix, and 15+ other platforms. On macOS/iOS/Android it can speak and listen; it also renders a live Canvas you control. The v2026.4.29 release made group chats agent-native (improved agent-to-agent communication), added follow-up commitment tracking, NVIDIA provider support, and the Codex harness plugin for substantially better code-agent performance.

## Key Features

- Runs on any OS; supports 20+ messaging channels from a single self-hosted gateway
- Voice in/out on macOS, iOS, Android; live Canvas rendering
- Group chat with multiple agents coordinating natively (as of v2026.4.29)
- Plugin system: Codex harness for coding tasks, skill extensions
- Own-your-data architecture — no cloud dependency, local-first
- 367k+ GitHub stars; TypeScript; sponsors include OpenAI, GitHub, NVIDIA, Vercel

## v2026.4.29 Release Highlights

- Group chats feel agent-native (major rework of agent communication)
- Follow-up commitments tracked from conversation context
- Safer exec mode, pairing controls, owner permission model
- NVIDIA provider + model catalogs added
- Faster startup; plugin/channel stability fixes

Anthropic's announcement of Claude Fable 5 and Claude Mythos 5 — the first models in the Claude 5 family, representing a new Mythos-class model tier that sits above Claude Opus in capability. Fable 5 is the generally available version with additional safety measures for dual-use capabilities; Mythos 5 shares the same underlying model but is available without those safety measures to approved organizations only.

The system prompt leaked around launch reveals a 120,000-character instruction set covering product information, refusal handling, tone/formatting rules, user wellbeing guidelines, an evenhandedness framework for political topics, and a persistent artifact storage API for cross-session key-value storage.

## Key Takeaways

- Fable 5 and Mythos 5 share the same underlying model; the difference is access controls and safety measures
- Claude now has beta products: Claude in Chrome (browsing agent), Claude in Excel, Claude in PowerPoint, and Claude Cowork (agentic knowledge-work desktop app)
- Artifacts can persist data across sessions via a storage API at a dedicated endpoint
- The system prompt explicitly instructs Claude not to foster over-reliance and never to encourage continued engagement
- Claude's reliable knowledge cutoff is end of Jan 2026; it uses web search proactively for post-cutoff events

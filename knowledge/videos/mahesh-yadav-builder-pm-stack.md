# Mahesh Yadav — How to Become a Builder PM

90-minute free episode from Aakash Gupta's podcast. Mahesh Yadav (AI PM at Amazon, Google, Meta, Microsoft) walks through the full builder PM stack.

## Topics covered (from tweet summary)

- Where to start with n8n and when to graduate past it
- How to actually use Claude Code (beyond day-one exploration)
- What to copy from OpenClaw for your own workflows
- Mac mini as a personal inference server: $599 M4 base, OpenClaw + Opus 4.6 + local Llama fallback, WhatsApp-connected, 24/7 runtime
- Delegation pattern: send a message → sandboxed agent runs 6-hour job → result returns to your phone

## Key argument

Builder PM = someone who can talk to customers, figure out what to build, ship v1, and reach 10 users without engineer help. The Mac mini + OpenClaw setup demonstrates the delegation pattern that enterprise AI agents (GCP, AWS) will productize in 2027.
